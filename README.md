# The NatiVis Server
## Install
### Prerequisites
* django
* requests
* django-cors-headers
* lxml
* nltk

**Note: For windows, lxml should be installed from [wheel](http://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml).**

## Running
```shell
python manage.py runserver 0.0.0.0:8002
```
**Note: Currently it is running on [http://166.111.139.15:8002]**
